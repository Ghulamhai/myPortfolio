# 🌐 Ghulam Haider - Portfolio Website

This is my personal portfolio website showcasing my academic background, projects, technical skills, and achievements.  
It’s designed to be modern, responsive, and minimal — built entirely with HTML, CSS, and JavaScript.

## 🚀 Live Demo
Visit: [https://ghulamhai.github.io/](https://ghulamhai.github.io/) *(or replace with your actual GitHub Pages link once deployed)*

---

## 📋 Features
- Responsive and mobile-friendly layout  
- Smooth scrolling navigation  
- Dynamic typing animation  
- Fade-in animations on scroll  
- Scroll-to-top button  
- Contact form with validation (frontend)  
- GitHub and LinkedIn integration links  

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Icons:** Font Awesome
- **Deployment:** GitHub Pages

---

## 📁 Project Sections
- **About Me** – A short intro and career background  
- **Education** – Academic milestones with CGPA  
- **Projects** – Featured academic and personal projects  
- **Skills** – Technical expertise grouped by category  
- **Achievements** – Competitive exams and recognitions  
- **Contact** – Email, phone, and social links
